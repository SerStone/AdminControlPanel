В проєкті реалізовано frontend і backend частини задані з ТЗ {{https://drive.google.com/file/d/11jn3rBK9C20Rg4CjqKFGal042m5cxVmx/view}} 
  -frontend {{https://github.com/SerStone/final-pr}} -- клонувати далі npm install та npm start після чого трішки почекати на завантаження компоненти 
  -backend {{https://github.com/SerStone/AdminControlPanel}} -- клонувати після чого в запуску Pycharm виставити venv та версію Python у мене була 3.12 далі ввести команду pip install -r requirements.txt 
  для запуску ввести python manage.py runserver. Також в корені AdminControlPanel присутня експортована колекція Postman Admin_Panel.postman_collection.json . 
  В якості БД використовувався MySQL http://owu.linkpc.net/mysql .
